# INT375: DATA SCIENCE TOOLBOX - Python Programming

This repository contains the study material and code implementations for the **INT375: Data Science Toolbox** course, organized by curriculum units.

## Syllabus Structure

### [Unit 01: Introduction to Python for Data Science](file:///home/vaibhav/Desktop/Programming/Python/Unit_01_Introduction_to_Python_for_Data_Science)
- **Topics**: Data collection basics, REST APIs (Weather, Pollution, Financial), Cloud Storage APIs (Google Drive/Sheets), Web Scraping basics.
- **Key Concepts**: GenAI API integration, Data extraction.

### [Unit 02: Data Manipulation with NumPy and Pandas](file:///home/vaibhav/Desktop/Programming/Python/Unit_02_Data_Manipulation_with_NumPy_and_Pandas)
- **Topics**: NumPy Arrays, Operations, Pandas Series & DataFrames, Data Cleaning, Handling Missing Data.
- **Library focus**: `numpy`, `pandas`, `polars`.

### [Unit 03: Data Visualization with Matplotlib and Seaborn](file:///home/vaibhav/Desktop/Programming/Python/Unit_03_Data_Visualization_with_Matplotlib_and_Seaborn)
- **Topics**: Principles of Visualization, Matplotlib Plots, Advanced visualization with Seaborn, Customizing plots.
- **Plots**: Bar, Scatter, Countplots, Customizations.

### [Unit 04: Exploratory Data Analysis (EDA)](file:///home/vaibhav/Desktop/Programming/Python/Unit_04_Exploratory_Data_Analysis_EDA)
- **Topics**: Summary Statistics, Correlation & Covariance, Outlier Detection (IQR, Z-Score), EDA on real datasets (Superstore).

### [Unit 05: Introduction to Statistical Analysis](file:///home/vaibhav/Desktop/Programming/Python/Unit_05_Introduction_to_Statistical_Analysis)
- **Topics**: Descriptive vs Inferential Statistics, Hypothesis Testing (Z-test, t-test, Chi-squared test), Probability Distributions.
- **Tests**: One-sample/Two-sample tests, Manual vs Library implementations.

### [Unit 06: Exploring the Role of Machine Learning](file:///home/vaibhav/Desktop/Programming/Python/Unit_06_Exploring_the_Role_of_Machine_Learning)
- **Topics**: ML Concepts, Supervised vs Unsupervised learning, CRISP-DM framework, Linear Regression, Classification.
